LiteGapps is an Custom Gapps for smartphones, tablet, desktop, free and open-source software. by providing complete, stable support and several additional features.

small friendly complete

LiteGapps first release on 12 july 2020 supports android Lollipop Up to Android 13. comes with 5 products namely : LiteGapps, LiteGapps++, LiteGapps Controller, LiteGapps Remover LiteGapps Addon

LiteGapps has system and systemless installation system. using 3 installation modes :

[MAGISK] = Magisk Module Flashable

[RECOVERY] = Recovery Module Flashable

[AUTO] = Magisk/Recovery Module Flashable

LiteGapps is made as light and minimal as possible but we also provide variants to meet the needs of the users

LiteGapps provides many Support,Version,Variant so that users can use LiteGapps according to their devices and needs. you can read the documentation so you don't download wrong !
